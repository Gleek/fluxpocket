FluxPocket
==========
[Pocket](http://getpocket.com) Integration for Fluxkraft
----------------------------------------------------------

**Project Under Development**

This project is done as the Google Summer of Code Project under Drupal.
You can read the full proposal on http://goo.gl/g5UpZY 

####Dependecies####
- fluxservices Integration
- duellsy pockpack Package \(install through composer\)

####What Works####
- Authentication

####What Does Not####
- Integration of Rules
- Basic Functionalities \(ADD/MODIFY/RETRIEVE etc.\) of the Pocket API
- Documentation
